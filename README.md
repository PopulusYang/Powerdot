# PowerDot

PowerDot 是一个基于 Java Swing 开发的简易演示文稿制作软件。它提供了类似于 PowerPoint 的基本功能，允许用户创建、编辑和播放幻灯片。

## 功能特性

* **幻灯片编辑**
  * 新建、删除、复制幻灯片页面。
  * 支持多种页面布局。
  * 所见即所得的编辑界面。

* **元素操作**
  * **文本框**：支持字体、大小、颜色设置，自动换行。
  * **基本图形**：直线、矩形、圆、椭圆。支持填充颜色、边框颜色、边框粗细调整。
  * **图片**：支持插入本地图片。
  * **交互**：支持元素的选中、移动、缩放、旋转。
  * **层级控制**：置于顶层、置于底层。

* **辅助功能**
  * **撤销/重做**：支持多步撤销和重做操作。
  * **剪贴板**：支持元素的复制、剪切、粘贴。
  * **文件操作**：支持保存工程文件 (.slide) 和打开现有工程。

* **放映与导出**
  * **幻灯片放映**：全屏播放，支持多种过渡动画（淡入淡出、滑动、缩放）。
  * **导出**：支持将当前页面导出为图片，或将整个演示文稿导出为 PDF 文件。

## 运行环境

* JDK 24 或更高版本。

## 如何运行

1. 编译项目源代码。
2. 运行 `PresentationApp` 类的主方法。

```bash
javac -encoding UTF-8 src/*.java
java -cp src PresentationApp
```
